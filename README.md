##  About

This project is a modern, responsive, and user-friendly **Stopwatch Web Application** developed as **Task 02** during my **SkillCraft Technology Web Development Internship**.

The application provides accurate time tracking with millisecond precision and allows users to start, pause, reset, and record lap times through an elegant glassmorphism-inspired interface.

---

##  Features

✔️ Start Stopwatch

✔️ Pause Stopwatch

✔️ Reset Stopwatch

✔️ Record Unlimited Lap Times

✔️ Millisecond Precision

✔️ Responsive Design

✔️ Modern Glassmorphism UI

✔️ Smooth Animations

✔️ Clean & Interactive User Experience
